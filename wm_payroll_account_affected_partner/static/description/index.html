Add Employee Partner to Payslip Journal Entry Items.

Features
  • Ability to configure adding partner to payslip journal entry items for any salary rules.
  • Ability to Enable/Disable this feature per salary rule
  • This feature only work if "Batch Account Move Lines" is disabled.
    • How it works?

    Salary Rule Configuration

    Odoo by default showing the partner in journal entry items for Net Salary rule and for any rules have a partner in partner field. This app will allow you to add payslip employee partner to journal entry item for any rule by enabling "Add Partner to Journal Item" in salary rule.
